# 🧪 Руководство по тестированию

## Запуск тестов

### Автоматические тесты валидации
```bash
node tests/moderation.test.js
```

**Что тестируется:**
- ✅ Валидация заголовков (длина, повторяющиеся символы, цифры)
- ✅ Валидация описаний (минимальная длина)
- ✅ Валидация цен (лимиты по типу объявления)
- ✅ Определение эмодзи

**Ожидаемый результат:**
```
✅ Passed: 18
❌ Failed: 0
📊 Total: 18
```

## Ручное тестирование

### Тест защиты от спама
1. Попробуйте создать объявление с заголовком "abc" → должна быть ошибка
2. Попробуйте создать объявление с заголовком "aaaaaaaa" → должна быть ошибка
3. Попробуйте создать объявление с ценой 100000€ → должна быть ошибка
4. Создайте 4 объявления подряд → 4-е должно заблокироваться (лимит 3/час)

### Тест защиты от ботов
Honeypot поле работает автоматически - боты, которые заполняют все поля формы, будут заблокированы.

## Когда запускать тесты

- ✅ Перед каждым деплоем
- ✅ После изменения функций валидации
- ✅ После добавления новых проверок
- ✅ При подозрении на баги

## Добавление новых тестов

Откройте `tests/moderation.test.js` и добавьте новый тест:

```javascript
test('название теста', () => {
  const result = функция(параметры);
  assert(result === ожидаемое, 'сообщение об ошибке');
});
```

## Troubleshooting

## Troubleshooting

**Тесты не запускаются:**
- Проверьте, что вы в корневой директории проекта
- Проверьте, что файл `lib/moderation.js` существует

**Тесты падают:**
- Проверьте, что не изменили сигнатуры функций
- Проверьте логи - там будет указано, какой тест упал

# 🚀 Полный чек-лист тестирования всего функционала (E2E)

Для автоматической проверки всего флоу (Лента -> Поиск -> Объявление -> Лайки -> Профиль -> Чат -> Аналитика) мы создали глобальный end-to-end тест:

```bash
npx playwright test tests/full-app-e2e.spec.ts --headed
```
С флагом `--headed` откроется реальный браузер и вы сможете своими глазами видеть, как робот "прокликивает" всю архитектуру (от поиска машин до создания объявлений и проверки графиков подписки). Убедитесь, что `npm run dev` запущен в другом окне терминала!

## Ручной прогон основных флоу (Manual QA)

Если вы хотите удостовериться вручную на устройстве, вот мастер-план проверок:

### 1. 🔐 Авторизация & Сессии
- [ ] Откройте Mini App из Telegram, убедитесь что вы залогинены и ваш аватар / имя подтянулись.
- [ ] Зайдите через обычный мобильный браузер по ссылке (вас должно перекинуть на `/login` с просьбой зайти через бота).

### 2. 🏠 Главная и Поиск
- [ ] Проверьте, что лента подгружает "Последние опубликованные".
- [ ] Кликните на любую категорию (например, 🧸 Детские). Убедитесь, что лента обновилась.
- [ ] Введите в поиск нетипичное слово (которого точно нет, например `абвгд123`), убедитесь, что вылез `EmptyState` ("Ничего не найдено").
- [ ] Перейдите во вкладку "Сохранённые поиски" в Поиске и Попробуйте сохранить текущий поиск.

### 3. 📝 Создание и Редактирование
- [ ] Перейдите во вкладку ➕ и заполните объявление (добавьте 2-3 фото из галереи).
- [ ] Установите нестандартную локацию маркером на карте.
- [ ] Нажмите "**Опубликовать**" -> Проверьте в консоли и в базе, что статус стал `active`.
- [ ] Откройте **Мои Объявления**, нажмите кнопку **Редактировать** на своем посте и измените цену. Сохраните. (Проверьте, что цена изменилась без создания дубля).

### 4. ❤️ Социальные механики и Карточка товара
- [ ] Откройте чужой пост. Нажмите на 🤍 (лайк). Перейдите в "Мои -> Избранное" и проверьте, что пост добавился. Уберите лайк — пост должен пропасть.
- [ ] Оставьте тестовый "Отзыв" продавцу в его профиле (⭐️⭐️⭐️⭐️⭐️). Обновите страницу и убедитесь, что рейтинг пересчитался.
- [ ] В профиле продавца нажмите "**Подписаться**". Опубликуйте тестовое объявление со своего 2-го аккаунта — вам в Telegram должен прилететь пуш от бота!

### 5. 💬 Внутренний чат (Conversations)
- [ ] В чужом посте нажмите "Написать". Напишите "Привет!".
- [ ] Проверьте во вкладке "Чаты", что создалась переписка.
- [ ] Попытайтесь написать `<script>alert(1)</script>` (тест XSS) — убедитесь, что в чате теги отрендерились просто как безопасный текст.

### 6. 📈 Статистика продавца
- [ ] Перейдите в `Мои -> Статистика`.
- [ ] Прокрутите вниз: вы должны увидеть 5 основных метрик и под ними **интерактивный график** (попробуйте поводить по нему пальцем/мышкой, дата должна всплывать).

### 7. 🛡️ Безопасность и Лимиты
- [ ] Быстро (10 раз за 20 секунд) нажимайте кнопку "Обновить профиль" или "Сохранить объявление" — вас должно отбросить с ошибкой (Rate Limit / Слишком много запросов).
